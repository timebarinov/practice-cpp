# Специализация 
## [Искусство разработки на современном C++](https://www.coursera.org/specializations/c-plus-plus-modern-development)
* **01. Разработка базы данных, которая работает с парами (дата, событие).**
* **02. Развитие базы данных, научив её выбирать и удалять события, удовлетворяющие заданному условию и рефакторинг.**
* **03. Создание своей поисковой системы.**
* **04. Разработка системы хранения транспортных маршрутов и обработки запросов к ней.**
* **05. Создания движка электронной таблицы, такой как Microsoft Excel или Google Spreadsheets.**
